# 🎮 NEON ARENA

A browser-based neon-style shooting game.

## 🚀 Features
- 2 Player Duel Mode
- Survival Mode
- Health system
- Neon UI design

## 🎮 Controls

### Game 1 (2 Player)
- Player 1: WASD + Space
- Player 2: Arrow Keys + Enter

### Game 2 (Survival)
- Move: WASD
- Aim: Mouse
- Shoot: Left Click

## 📁 Tech Used
- HTML
- CSS
- JavaScript

## ▶️ Run
Open `index.html` in browser
